# grammar/

Purpose
- Source-of-truth grammar assets and exported forms used to build the parser and external tooling.

Key assets
- grammar.233, grammar.300: Publication baselines (2nd/3rd) used as inputs to `tools/mkgramy`.
- gmiddle.y, gmiddle.glr.y: Middle sections merged into the generated `grammar.y` (GLR variant adds `%glr-parser`).

Generated by Make targets
- grammar.y: Yacc/Bison input, produced by `make regen` via `tools/mkgramy`.
- grammar.ebnf: EBNF export via `make ebnf`.
- grammar.lark: Lark-compatible EBNF via `make lark`.

Notes
- `BASELINE=233|300` selects the baseline; `GLR=1` switches to GLR middle.
- Do not edit generated files (`grammar.y`, `grammar.ebnf`, `grammar.lark`) by hand; re-run generation instead.
